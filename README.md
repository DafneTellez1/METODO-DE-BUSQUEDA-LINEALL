# METODO-DE-BUSQUEDA-LINEALL
## Análisis de complejidad

Para mí, la búsqueda lineal es una de las herramientas mas honestas y directas que tenemos. Es literal como buscar algo en un cuarto desordenado, vas objeto por objeto hasta que das con lo que quieres. No necesita trucos ni que los datos estén acomodados de cierta forma, simplemente recorre la lista del inicio al final.

## Casos de uso
En general, el metodo de busqueda lineal sirve como nuestra herramienta de confianza para explorar datos sin restricciones. Su función principal es garantizarnos que podemos encontrar cualquier información, sin importar qué tan desordenada y larga esté la lista.

## Comparativa

A diferencia de otros métodos que te exigen reglas las cuales necesite que todo este ordenado, la búsqueda lineal es la solución universal. Si el dato existe, lo va a encontrar. Es ideal para cuando manejamos archivos pequeños, listas que cambian constantemente o cuando simplemente no queremos gastar recursos del procesador ordenando miles de datos solo para una búsqueda rápida. Básicamente, es el metodo que nos asegura que el programa siempre tendrá una respuesta, incluso en medio del desorden.
