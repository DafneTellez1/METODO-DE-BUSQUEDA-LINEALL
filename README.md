# METODO-DE-BUSQUEDA-LINEALL
#Análisis de complejidad#
Para mí, la búsqueda lineal es la herramienta más honesta y directa que tenemos. Es literal como buscar algo en un cuarto desordenado: vas objeto por objeto hasta que das con lo que quieres. No necesita trucos ni que los datos estén acomodados de cierta forma; simplemente recorre la lista del inicio al final. Es la definición perfecta de 'fuerza bruta' en programación porque no asume nada del archivo, solo se pone a trabajar hasta encontrar el resultado.

##Casos de uso
En general, el método de búsqueda lineal sirve como nuestra herramienta de confianza para explorar datos sin restricciones. Su función principal es garantizarnos que podemos encontrar cualquier información, sin importar qué tan desordenada o caótica esté la lista.

##Comparativa
A diferencia de otros métodos que te exigen reglas (como que todo esté ordenado), la búsqueda lineal es la solución universal: si el dato existe, lo va a encontrar. Es ideal para cuando manejamos archivos pequeños, listas que cambian constantemente o cuando simplemente no queremos gastar recursos del procesador ordenando miles de datos solo para una búsqueda rápida. Básicamente, es el método que nos asegura que el programa siempre tendrá una respuesta, incluso en medio del desorden.
